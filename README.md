# HuntGround

Static prototype for a Canada and US hunting marketplace and planning app.

## Run locally

Open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server 4173
```

Then visit:

```text
http://localhost:4173/index.html
```

## Included

- Login and signup demo pages
- Guided marketplace journeys
- Top search panel with filters
- Map and list result views
- Offer creation panel
- Trip checklist
- Admin panel
